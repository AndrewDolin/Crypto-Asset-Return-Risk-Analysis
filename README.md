# Crypto-Asset-Return-Risk-Analysis
This project analyzes the daily returns of Ethereum, Bitcoin, Dollars, and other currencies and evaluates risk metrics commonly used in quantitative finance.

Risk Metrics Implemented:
- Annualized Volatility
- Sharpe Ratio
- Maximum Drawdown
